# JWT Authentication Project

## 🚀 Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT (JSON Web Token)

## 🔐 Features
- User Login & Registration
- JWT Token Generation
- Secure API Endpoints
- Token Validation

## ▶️ How to Run
1. Clone repo
2. Run Spring Boot application
3. Test using Postman

## 📌 API Endpoints
- /login
- /register
- /secure-data
